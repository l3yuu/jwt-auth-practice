Project Name: JWT Authentication Practice

Description
A full-stack authentication system demonstrating secure session management. This project implements JSON Web Tokens (JWT) stored in HttpOnly cookies to protect against Cross-Site Scripting (XSS).

Key Features

User registration with password hashing via bcryptjs.

Secure login with JWT issuance.

Persistent sessions using MongoDB Atlas.

Protected routes on the frontend using React Router and Axios.

Automated authentication check on page mount.
